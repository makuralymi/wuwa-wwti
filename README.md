# WWTI 鸣潮共鸣测试

一个仿照 `../SBTI-test` 结构实现的纯静态娱乐测试页。

- 入口: `index.html`
- 内容: 16 道鸣潮主题题目, 多标签计分, 14 个结果类型
- 运行: 可直接打开 `index.html`, 也可以用本地静态服务器访问
- 部署: 执行 `scripts/deploy.sh`, 同步到 `/var/www/ygti` 并 reload nginx
- 图鉴达成规则: `docs/type-unlock-rules.md`
- 配图: 使用本地 `assets/openmoji/` 里的 OpenMoji SVG 图标组合成各人格卡面
- 出题: 每次开始测试时会随机打乱题目顺序和题内选项顺序

本项目为粉丝向娱乐页面。鸣潮相关角色、术语与世界观属于其权利方。
